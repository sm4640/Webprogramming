1. 힘짜속

- 기간: 4월 초 ~ 5월 25일
- 사용했던 기술: 프론트 -> react / 백 -> django / DB -> SQLite
- 서비스 설명: 감정 쓰레기통 서비스

1. 힘든 일, 짜증나는 일, 속상한 일을 선택하고
2. 편지지에 자신의 감정을 적은 후
3. 쓰레기통에 버리면
4. 포춘쿠키가 운세를 전해주듯, 작은 위로의 말 건네주기

2. 물어봐

- 기간: 7월 중순 ~ 8월 19일
- 사용했던 기술: 프론트 -> react.js, next.js / 백 -> django-restframework / DB -> SQLite
- 서비스 설명: 디지털 격차 극복을 위한 위치기반 멘토멘티 매칭 서비스

1. 앱 첫 실행 시, 멘토/멘티 설정
2. 멘티는 카테고리 선택 후 멘토에게 도움 요청
3. 멘토는 들어온 요청 확인 뒤 수락
4. 멘토가 수락하면 멘티는 멘토의 도착시간 파악 가능
5. 멘토는 멘티의 위치 파악 가능
6. 도움을 받고 난 후, 멘티는 멘토의 이용후기를 작성할 수 있음

3. 세피로트 (최우수상 수상🏆)

- 기간: 10월 말 ~ 11월 12일
- 사용했던 기술: 프론트 -> react.js / 백 -> django-restframework / DB -> MySQL
- 서비스 설명: 친환경 광고를 보고 나무를 키워서 실제로 심는 서비스

1. 자신의 나무에 닉네임 설정 가능
2. 친환경 관련 제품 광고를 보면 물을 줄 수 있음
3. 최대 레벨이 되면 실제로 나무를 심을 수 있음
4. 심어놓은 나무를 추후에도 모니터링 가능

4. 위벤트

- 기간: 9월 초 ~ 진행 중
- 사용했던 기술: 프론트 -> react.js / 백 -> django-restframework / DB -> MySQL
- 서비스 설명: 행사 주최자와 협찬사를 편리하게 매칭시켜주는 서비스

1. 주최자는 서비스에서 정해놓은 양식대로 행사 정보에 대해 입력하고 등록함
2. 협찬사는 다양한 행사 정보를 통일된 양식으로 쉽게 접할 수 있음
3. 관심있는 행사 주최자에게 바로 채팅 기능을 통해 연락할 수 있음
4. 주최자와 협찬사가 협찬 내용 조율 후 협찬 체결
